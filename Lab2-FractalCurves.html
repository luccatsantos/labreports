2. Random Walk with Random Step Length

In a traditional random walk, the movement follows a fixed step length, which, while introducing directional randomness, still maintains a predictable pattern in terms of distance traveled per step. The modification enhances this by introducing a randomly varying step length, where each step is chosen from a range between min_step_length and max_step_length. This additional layer of randomness makes the motion more unpredictable and closely resembles real-world Brownian motion, where particle movements are influenced by varying external forces. By allowing the step size to change dynamically, the walk becomes more organic, avoiding the uniformity of a fixed-step approach and creating a more natural and realistic simulation of random movement. 


	import random

	# Step 1: Initialize the turtle canvas
	canvas_width = 1000
	canvas_height = 800
	initializeTurtle(initial_window_size=(canvas_width, canvas_height), mode='auto')

	# Step 2: Function to perform a random walk (Brownian motion)
	def random_walk(steps, min_step_length, max_step_length):
    		speed(0)  # Set speed to maximum for faster drawing
    		penup()
    		jump(450, 450)  # Start near the middle of the canvas
    		pendown()

    	for _ in range(steps):
        	step_length = random.uniform(min_step_length, max_step_length)  # Random step length
        	forward(step_length)  # Move forward by the random step length
        	right(random.uniform(-180, 180))  # Turn by a random angle between -180 and 180 degrees

	# Step 3: Execute the random walk with increased steps and/or step length
	steps = 1000
	min_step_length = 5
	max_step_length = 15

	random_walk(steps, min_step_length, max_step_length)

	# Step 4: Display the result
	show()

The image shows the result of a random walk simulation, where a turtle follows a series of randomly determined steps and directions, creating a chaotic yet natural path. The jagged, unpredictable trajectory demonstrates the impact of both directional randomness and varying step lengths, resembling phenomena such as particle diffusion or animal foraging patterns. The clustering in some areas suggests sections where shorter steps accumulated, while longer steps contributed to more dispersed movement. This visualization highlights the power of stochastic processes in simulating real-world randomness.

3. Drawing the Cantor Set

​The Cantor set, introduced by German mathematician Georg Cantor in 1883, is a classic example of a fractal in mathematics. It is constructed by repeatedly removing the middle third of a line segment. This process leaves a set of points that is uncountably infinite yet has zero measure, meaning its total length sums to zero. The Cantor set is nowhere dense, containing no intervals, and is self-similar, as each part resembles the whole. Cantor's work on this set was foundational in the development of modern set theory and introduced concepts of different sizes of infinity. ​ 


# Define the cantor_set function
def cantor_set(length, order):
    if order == 0:
        forward(length)
    else:
        cantor_set(length/3, order - 1)
        penup()
        forward(length/3)
        pendown()
        cantor_set(length/3,order-1)

initializeTurtle()

# Draw cantor_sets of different orders
jump(50, 400)
face(90)
color('red')
cantor_set(800, 1)  # Order 1

jump(50, 350)
face(90)
color('blue')
cantor_set(800, 2)  # Order 2

jump(50, 300)
face(90)
color('orange')
cantor_set(800, 3)  # Order 3

jump(50, 250)
face(90)
color('pink')
cantor_set(800, 4)  # Order 4

jump(50, 200)
face(90)
color('purple')
cantor_set(800, 5)  # Order 5

show()

This Python script uses the turtle graphics module to recursively draw the Cantor set. The function takes two parameters: length, the length of the current segment, and order, the recursion depth. At each recursion level, it draws the left third of the segment, skips the middle third, and then draws the right third, calling itself recursively to achieve the fractal pattern. The script initializes the turtle and then draws Cantor sets of increasing orders (from 1 to 5 in this example), each in a different color and at a different vertical position, to visualize the fractal's progression.​
