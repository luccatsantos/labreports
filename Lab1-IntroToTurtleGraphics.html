1. Introduction

In this lab, I designed a Malta Cross using a turtle-based drawing tool and extended it into a fractal-inspired pattern. The experiment demonstrates the iterative nature of fractals.

2. Original Malta Cross

The initial code generated a simple Malta Cross using a turtle graphics library. The background was set to red, and the cross is drawn in white. The key features of this code include: 1) Initialization of the turtle; 2) Positioning the turtle appropriately; 3) Iterative drawing of the cross using loops. 



	# Initialize the turtle
	initializeTurtle()
	showturtle()
	color('white')
	bgcolor('red')

	# Move to a suitable starting position
	jump(500, 300)  # Adjust these values as needed to center the square
	face(15)  # Ensure the turtle is facing up

	#Drawing a Malta Cross
	d=100
	jump(700, 300)
	face(-10)

	for _ in range(4):
    		forward(d)
    		right(150)
    		forward(d/3)
    		left(100)
    		forward(d/3)
    		right(150)
    		forward(d)
    		left(110)

	jump(0,0)        
	show()

The Malta Cross is constructed using a sequence of forward and rotational movements. The logic involves four repetitions to construct each arm of the cross. The turn angles (150, 100, and 110 degrees) define the shape of the arms and their symmetry.

3. Extending the Code: Fractal Spiral Malta Cross

I extended the original code to create a fractal spiral Malta Cross pattern. This was achieved by: 1) Wrapping the original loop inside an additional loop to introduce fractal-like repetition; 2) Adjusting the turtle’s position dynamically after each iteration; 3) Reducing the segment length iteratively to create a shrinking effect; 4) Introducing a termination condition to prevent infinite recursion. 



	# Initialize the turtle
	initializeTurtle()
	showturtle()
	color('white')
	bgcolor('red')

	# Move to a suitable starting position
	jump(500, 300)  # Adjust these values as needed to center the square

	#Drawing a Malta Cross

	d=300

    	  forward(d/3)
    	  right(150)
    	  forward(d)

  	jump(getx()*1.000001, gety()*1.000001)
  	d=d-20
  	right(5)
  	if d <= 0:
    	 break  # Exit the loop completely
	jump(0,0)        
	show()

By iterating multiple times and shrinking the segment length, the result formed a fractal-like spiral pattern. The key observations include: The fractal emerges naturally through self-replication at decreasing scales. The small rotational offset (5°) adds a twisting effect, contributing to a spiral structure.
