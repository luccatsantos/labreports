In this lab, we designed a Malta Cross using a turtle-based drawing tool and extended it into a fractal-inspired pattern. The experiment demonstrates the iterative nature of fractals.

2. Original Malta Cross

The initial code generated a simple Malta Cross using a turtle graphics library. The background was set to red, and the cross is drawn in white. The key features of this code include: - Initialization of the turtle; - Positioning the turtle appropriately; - Iterative drawing of the cross using loops. 



	# Initialize the turtle
	initializeTurtle()
	showturtle()
	color('white')
	bgcolor('red')

	# Move to a suitable starting position
	jump(500, 300)  # Adjust these values as needed to center the square
	face(15)  # Ensure the turtle is facing up

	#Drawing a Malta Cross
	d=100
	jump(700, 300)
	face(-10)

	for _ in range(4):
    		forward(d)
    		right(150)
    		forward(d/3)
    		left(100)
    		forward(d/3)
    		right(150)
    		forward(d)
    		left(110)

	jump(0,0)        
	show()

The Malta Cross is constructed using a sequence of forward and rotational movements. The logic involves four repetitions to construct each arm of the cross. The turn angles (150, 100, and 110 degrees) define the shape of the arms and their symmetry.
