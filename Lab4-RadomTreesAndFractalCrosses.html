1. Random Fractal Trees

The original fractal tree function follows a deterministic pattern, reducing the branch length by a fixed amount (shorten_by) and maintaining a constant branching angle. This creates a symmetrical and predictable structure. I've one modification to introduce randomness in both the branch length reduction and the branching angle. By using random.uniform(), each iteration produces a slightly different tree shape, mimicking natural variations found in real trees. This randomness makes the tree appear more organic and less rigidly structured. The modification aligns with Mandelbrot’s concept of fractional Brownian motion, where natural patterns exhibit self-similarity with an element of randomness, as seen in mountain ranges and coastlines (Mandelbrot, 1983). By integrating randomness, the new function better represents the stochastic nature of real-world fractal structures. 


  # Define the random fractal tree function
  def build_random_tree(branch_length):
    if branch_length > 5:

        forward(branch_length)

        new_length = branch_length - random.uniform(branch_length/6, branch_length/3)

        angle = random.uniform(20,50)
        
        left(angle)

        build_random_tree(new_length)

        right(angle * 2)

        build_random_tree(new_length)

        left(angle)

        backward(branch_length)

  # Initialize the turtle
  initializeTurtle(initial_window_size=(800, 800))

  # Set up the turtle's starting position and orientation
  jump(400, 600)  # Starting position at the bottom center of the canvas
  face(0)
  color('green')  # Set the pen color

  # Draw the fractal tree
  build_random_tree(100)

  # Display the drawing
  show()

Due to the randomness in the code, no two trees will be identical, though they retain an overall self-similar structure. The left tree appears denser with more intricate branching, while the right tree has a slightly more open and varied distribution of branches, highlighting the impact of stochasticity in fractal growth.
