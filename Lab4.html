Background

The concept of a fractal is tied to self-similarity and potentially infinite complexity, meaning that zooming in reveals finer and finer repeating structures. One of the most iconic fractals is the Mandelbrot set, defined by the iteration:

$$ z_{n+1} = z_n^2 + c, $$ with $z_0 = 0.$ A complex number $c = x + iy$ lies in the Mandelbrot set if the sequence $(z_n)$ remains bounded for all $n$. In practice, we assume it remains bounded if $|z_n| \le 2$ for all iterations $n < N_{\mathrm{max}}.$

The Mandelbrot set is a cornerstone of complex dynamics, exhibiting a richly detailed boundary where one can zoom in infinitely and continue to find new structures, often reminiscent of the overall shape.

Methods / Algorithm

Below is our JIT-optimized Python code that computes and plots both a broad view and a zoomed-in region of the Mandelbrot set. Numba is used to speed up the repeated iteration checks. 

import numpy as np
import matplotlib.pyplot as plt
from numba import jit

# JIT-optimized mandelbrot function
@jit(nopython=True)
def mandelbrot(c, max_iter):
    z = 0
    n = 0
    while abs(z) <= 2 and n < max_iter:
        z = z*z + c
        n += 1
    return n

# JIT-optimized mandelbrot set generator
@jit(nopython=True)
def mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter):
    r1 = np.linspace(xmin, xmax, width)
    r2 = np.linspace(ymin, ymax, height)
    n3 = np.empty((width, height))

    for i in range(width):
        for j in range(height):
            n3[i, j] = mandelbrot(r1[i] + 1j * r2[j], max_iter)

    return n3

# Parameters for the plot
xmin, xmax, ymin, ymax = -2.0, 1.0, -1.5, 1.5
width, height = 800, 800
max_iter = 100

# Generate the Mandelbrot Set (broad view)
n3 = mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter)

# Now set parameters for a zoomed-in region
xmin, xmax, ymin, ymax = -0.75, -0.74, 0.1, 0.11  # Zoom into this specific area
width, height = 800, 800  # Resolution of the image
max_iter = 1000  # Increased number of iterations

# Generate the Mandelbrot Set with zoom
n3 = mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter)

# Plot the zoomed Mandelbrot Set
plt.figure(figsize=(10, 10))
plt.imshow(n3.T, extent=[xmin, xmax, ymin, ymax], cmap='hot')
plt.colorbar()
plt.title("Mandelbrot Set - Zoomed In")
plt.show()

Discussion

Using this JIT-optimized approach greatly speeds up the iterative calculations needed to classify points in the set. The Mandelbrot set itself highlights fundamental properties of complex quadratic polynomials: small changes in the initial parameter $c$ can lead to drastic differences in convergence or divergence. The zoomed region showcases the fractal’s infinite detail and how parameter tuning (like increasing max_iter) helps reveal more intricate structures.

In practice, generating very deep zooms can become computationally expensive. Techniques such as arbitrary precision arithmetic, adaptive iteration strategies, and parallel/GPU-based solutions can push these explorations to extreme magnifications, unveiling ever more complex filigree along the Mandelbrot boundary.
